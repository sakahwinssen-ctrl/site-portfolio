SAÉ 15

Traitement de données

Cette SAÉ m’a permis de travailler seul sur un script Python capable de récupérer, traiter et produire des données exploitables.

Retour aux SAÉ

Présentation de la SAÉ

Le but était de développer un script Python capable d’interroger l’API Overpass d’OpenStreetMap, de télécharger des données sur des points d’intérêt et de générer automatiquement des fiches structurées. J’ai réalisé cette SAÉ seul, ce qui m’a demandé de comprendre le sujet, d’écrire le code et de corriger les erreurs de manière autonome.

Mes tâches et mes preuves

J’ai travaillé dans VS Code avec Python. J’ai commencé par comprendre la structure des données JSON renvoyées par l’API, puis j’ai écrit des fonctions pour récupérer un élément OSM, extraire ses tags, trier les informations utiles et produire un fichier Markdown lisible.

  • Connexion à l’API Overpass / OpenStreetMap.
  • Récupération et lecture de données JSON.
  • Extraction du nom, du type, des coordonnées et des tags.
  • Tri des informations utiles.
  • Génération automatique d’un rendu Markdown.

Impact et utilité de la preuve

Ces deux captures montrent les parties les plus importantes de mon script. La première partie sert à récupérer les informations depuis l’API OpenStreetMap au format JSON. La deuxième transforme ces données en contenu Markdown lisible. Cette preuve montre que j’ai utilisé la programmation pour automatiser un traitement concret : récupérer des données, les trier, puis générer un rendu exploitable.

Auto-évaluation

Comme j’ai réalisé cette SAÉ seul, j’ai dû beaucoup chercher par moi-même. La difficulté principale a été de comprendre la structure des données JSON et de gérer les cas où certaines informations n’existaient pas. Cette SAÉ m’a permis de gagner en autonomie en Python, mais je dois encore progresser pour rendre mon code plus propre, plus court et mieux commenté.
